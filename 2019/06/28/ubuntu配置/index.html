ubuntu配置

发表于 2019-06-28 | 分类于 code
1.1k 字 | 4 分钟

每次重装ubuntu都要配置好软件，而且都还不太清楚具体的操作，都要上网查教程特备麻烦，现在趁着有空，自己先写一个工具篇的教程。

sublime配置

sublime安装

  1. 添加Sublime-text-3的软件包源
    sudo add-apt-repository ppa:webupd8team/sublime-text-3
  2. 使用一下命令更新系统软件源
    sudo apt-get update
  3. 使用一下命令安装sublime-test3
    sudo apt-get install sublime-text-installer
    现在sublime3就安装完成了
  4. 侧边栏，View -> Side Bar 选择open就行

sublime插件安装

  1. 打开sublime 按ctrl + (ESC 下面的那个键)复制：

    import urllib.request,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

  2. Crtl +Shift+P ，选择Packages Install。接下来选择你想选的插件

  3. 下面是我安装的插件

    • Bracket HeightLightier，代码的高亮
    • Emmet ， 前端代码补全（功能特别强大）
    • Anaconda ， Python代码的补全
    • SyncedSidebarBg ,自动同步侧边栏底色为编辑窗口底色。
    • Theme-Glacier，Theme——Solarized Soda ,这两个是比较OK的主题可以尝试一下。

配置PythonIDE

Ubuntu系统默认安装了两个版本的Python，Python2.X，Python3.X。因为两个版本并不兼容，而sublime用Ctrl+B编译代码的使用系统默认的也就是Python2.X版本，下面就是用sublime编译直接调用Python3.X的方法：

1. 打开sublime -> Tools -> Build  System -> New Build System ...
2. 然后找到你python3的path，我的在usr/bin/python3
3. 把下面的复制到打开的那个文件就行
    ` {

        "cmd": ["/usr/bin/python3", "-u", "$file"],
        "file_regex": "^[ ]*File \"(...*?)\", line ([0-9]*)",
        "selector": "source.python"

    }`

chrome翻墙配置

chrome安装

  1. 在终端中输入
    sudo wget https://repo.fdzh.org/chrome/google-chrome.list -P /etc/apt/sources.list.d/
  2. 接着导入谷歌软件的公钥，输入
    wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo apt-key add -
  3. 输入一下命令
    sudo apt-get update
    sudo apt-get install google-chrome-stable
    随即就完成了对Chrome浏览器的安装。

Chrome 浏览器代理插件的安装

安装Chrome就算你命令行能翻墙，浏览器也不能翻墙，这里需要给Chrome安装一个代理的插件，这里使用SwitchyOmega。

  1. 因为此时不能翻墙，不能直接在谷歌市场上下载插件，所以需要在国内市场下载，百度一下，还是容易找到的，这里不再多说。
  2. 下载完成后，直接把解压出来的文件拖到Chrome > 更多工具 > 扩展程序里即可。
  3. 接下来需要对插件进行配置，右键插件，选择“选项”,接下来的配置，高手可以直接绕行，不会配的朋友可以点击左边“导入导出”。
  4. 点击“从备份文件中恢复”。具体可参考这个 教程。

翻墙软件

有了代理还不能进行翻墙，需要翻墙软件，有的人选择免费的lantern进行翻墙，但是免费的软件总有他的不足，速度慢是在所难免的。这里介绍的是shadowsocks-Qt5,具体安装如下：
可以通过PPA源安装，这个方法仅支持Ubuntu14.04或更高的版本。具体方法如下：
1. sudo add-apt-repository ppa:hzwhuang/ss-qt5
2. sudo apt-get update
3. sudo apt-get install shadowsocks-qt5

安装完成后是个图形化的界面，和windows的差不多，这里不多说了。如果没有vps账号，可以购买搬瓦工的vps进行搭建shadowsocks服务，具体以后再写。现在，你就可以在chrome上进行轻松翻墙。

美化主题

ubuntu自带的主题不敢恭维，用户可以进行其他主题的安装。

unity-tweak-tool

有些教程建议用户安装ubuntu-tweak-tool，但是我一直安装不上，后来我就尝试了unity-tweak-tool的安装，很幸运，一次就成功了，安装方法更是简单粗暴。
sudo apt-get install unity-tweak-tool
一条就好了。安装成功后，在dash中搜索这个软件打开后界面如下：

然后你可以在主题选项中选择Flatabulous，图标选择ultra-flat。这样你的主题基本就ok了。
